# Java OpenGL Exercises (TpOgl)

A collection of computer graphics exercises using Java and OpenGL (JOGL)

## Project Structure

This project contains four main exercise modules:

### 1. **OpenGL Basics** (`org.ogl`)
- Basic OpenGL primitives and drawing operations
- Window management with AWT and OpenGL contexts
- Foundational graphics programming concepts

### 2. **2D Forms** (`org.forms`)
- 2D geometric shapes and primitives
- Includes: Box, Cone, Cylinder, Sphere, Torus variants, Dome, Wedge
- Shape rendering and manipulation exercises

### 3. **Polygons** (`org.polygone`)
- Polygon operations and algorithms
- Point-in-polygon tests
- Segment intersection
- Convex hull operations
- Polygon classification and analysis

### 4. **3D Volumes** (`org.volumes`)
- 3D geometric volumes and solids
- Colored primitives: ColorCube, ColorCone, ColorCylinder
- 3D transformations and rendering

### 5. **Solid Foundations** (`org.fond`)
- Advanced solid modeling
- Furniture models: Chair, Table, Fan
- Complex primitives and composite objects
- Observer pattern implementation for graphics

## Technology Stack

- **Java** - Core programming language
- **JOGL (Java OpenGL)** - OpenGL binding for Java
- **AWT** - Abstract Window Toolkit for GUI
- **Maven** - Build and dependency management

## Getting Started

1. Ensure you have Java JDK installed
2. Install Maven (or use IDE's built-in Maven)
3. Clone or download this project
4. Run `mvn install` to install dependencies
5. Navigate to any exercise module and run its `Exo.java` file

## Running Exercises

Each module has its own `Exo.java` entry point:
- `org.ogl.Exo` - Basic OpenGL exercises
- `org.forms.Exo` - 2D forms exercises  
- `org.polygone.Exo` - Polygon algorithms
- `org.volumes.Exo` - 3D volume exercises
- `org.fond.Exo` - Solid modeling exercises
